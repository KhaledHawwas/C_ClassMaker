# C_ClassMaker
create header file and source file of c++ language by Java <br>
### How to use
in member field write type , name and  **g, s, p** characters (or not)<br>
**g** for **get** **s** for **set** **c** for **constructor**<br>
notes: type cannot be more than word<br>
